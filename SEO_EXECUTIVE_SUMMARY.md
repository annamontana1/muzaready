## TECHNICAL SEO CHECKLIST

### Critical (Must Have Before Launch)

- [ ] **URL Structure:** `/vlasy-k-prodlouzeni/[kategorie]/[product-slug]/` (with length)
- [ ] **Title Tags:** Auto-generated per pattern (50-60 chars)
- [ ] **Meta Descriptions:** Auto-generated per pattern (150-160 chars)
- [ ] **H1 Tags:** One per page, keyword-focused
- [ ] **Schema Markup:** Product, Breadcrumb, Organization, FAQ
- [ ] **robots.txt:** Block /admin/*, /orders/*, allow everything else
- [ ] **sitemap.xml:** Dynamic, includes all public pages, submitted to GSC
- [ ] **Canonical Tags:** Prevent duplicate content on filtered pages
- [ ] **Mobile-First:** Responsive design, touch-friendly
- [ ] **Core Web Vitals:** All Green (LCP < 2.5s, FID < 100ms, CLS < 0.1)
- [ ] **Lighthouse Score:** 95+/100
- [ ] **No Crawl Errors:** 0 404s, 503s, timeouts in GSC

### High Priority (Within 2 Weeks)

- [ ] **Internal Linking:** 2-5 contextual links per page
- [ ] **Image Optimization:** Compressed, WebP format, descriptive alt text
- [ ] **OG Images:** 1200x630px for social sharing
- [ ] **Structured Data:** All product pages have JSON-LD schema
- [ ] **Analytics:** GA4 with purchase/add-to-cart events firing
- [ ] **Heatmaps:** Hotjar or Clarity for user behavior tracking
