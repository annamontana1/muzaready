## Executive Summary

Múza Hair má řidší pozici jako jediný český výrobce prodloužených vlasů s vlastním barvírnou. Tato SEO strategie mapuje 250+ cílových klíčových slov na konkrétní stránky, koordinuje úpravu URL struktur s vývojem katalogu a zavádí schema markup pro e-commerce. Roadmap je zarovnán s Development Priorities 1-6 (8-10 dny), zajišťující, že SEO foundations budou připraveny ještě před spuštěním košíku a implementací plateb. Fokus je na Czech e-commerce best practices, zdůraznění originality značky (českého výrobce) a konverzní optimalizaci pro vlasový market.

### 2.1 URL Struktura

#### Pravidlo 1: Homepage
```
URL: /
Keywords: Top 5 brand + top category keywords
```

#### Pravidlo 2: Kategoriální Stránky
```
URL: /vlasy-k-prodlouzeni/[kategorie]/
Struktura:
- /vlasy-k-prodlouzeni/nebarvene-panenske/
- /vlasy-k-prodlouzeni/barvene-vlasy/

Délka v URL pro SEO:
- /vlasy-k-prodlouzeni/nebarvene-panenske-40cm/
- /vlasy-k-prodlouzeni/luxe-60cm/
- /vlasy-k-prodlouzeni/standard-45cm/
```

#### Pravidlo 3: Produktové Stránky (PDP)
```
URL: /vlasy-k-prodlouzeni/[kategorie]/[product-slug-with-length]/
Příklady:
- /vlasy-k-prodlouzeni/nebarvene-panenske/luxe-60cm-odstin-1/
- /vlasy-k-prodlouzeni/barvene-vlasy/standard-45cm-blond/
- /vlasy-k-prodlouzeni/nebarvene-panenske/platinum-70cm-odstin-2/

Slug Formát:
[tier]-[lengthCm]cm-[shade-descriptor]
Důvod: Zvýšuje SEO relevanci délky přímo v URL
```

#### Pravidlo 4: Blogové Stránky
```
URL: /blog/[slug]/
Příklady:
- /blog/jak-vybrat-spravnou-delku-vlasu/
- /blog/pece-o-prodlouzene-vlasy-10-pravidel/
- /blog/standard-vs-luxe-vs-platinum/
- /blog/jak-dlouho-vydrzi-prodlouzene-vlasy/
```

#### Pravidlo 5: Kategoriální Filtry (bez duplikátu!)
```
DOBRÝ PŘÍSTUP (Faceted Navigation):
/vlasy-k-prodlouzeni/nebarvene-panenske/?length=40cm&tier=luxe

ŠPATNÝ PŘÍSTUP (Duplicate Content):
/vlasy-k-prodlouzeni/nebarvene-panenske/luxe/40cm/  ←Dup!
/vlasy-k-prodlouzeni/nebarvene-panenske-luxe-40cm/  ← Dup!

Implementace: rel="canonical" na base kategoriální straně
```

